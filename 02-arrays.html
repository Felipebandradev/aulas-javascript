Arrays (Vetores e Matrizes)

Conceito

Vetor (ou array) é uma estrutura de dados que permite armazenar em memoria uma listas de informações.Pode ser declarado como varíavel e/ou constante, e cada um dos dados/informações armazenados possui uma referência chamda Indices.

Array de uma unica dimenssão é chamado de Vetor, enquanto que com duas ou mais dimenções, é chamado de Matiz.
